Running telnetlogger on my home IP

Wed Nov 02, 2016 by cmohn in  Virtualization

Robert Graham of erratasec has created a small honeypot tool called telnetlogger. This is a simple program to log login attempts on Telnet (port 23). It's designed to track the Mirai botnet. Right now (Oct 23, 2016) infected Mirai machines from around the world are trying to connect to Telnet on every IP address about once per minute. This program logs both which IP addresses are doing the attempts, and which passwords they are using. Read more →
