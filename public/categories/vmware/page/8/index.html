VMware Labs has released a new fling called Makyo (魔境 _makyō_ means “ghost cave” or “devil’s cave.”) which basically is a tool to copy VMs or vApps from one vCenter instance to another. It works by doing an automated OVF export on the source vCenter, and then an import on the destination vCenter.
