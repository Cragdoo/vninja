Place the host in maintenance mode, then run the following command to do an online update to ESXi 5.5: [cc lang=«bash» width=«100%» theme=«blackboard» nowrap=«0»] ~ # esxcli software profile update -d https://hostupdate.vmware.com/software/VUM/PRODUCTION/main/vmw-depot-index.xml -p ESXi-5.
