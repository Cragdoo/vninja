Quick and Dirty ESXi 5.5 Upgrade

Mon Sep 23, 2013 by cmohn in  Virtualization  5.5, Command Line, esxcli, ESXi, Host, Upgrade, VIB, VMware, vSphere

As I´ve posted about earlier, you can update your ESXi hosts to a new release from the command line. Now that ESXi 5.5 has been released, the same procedure can be applied to upgrade once more. Place the host in maintenance mode, then run the following command to do an online update to ESXi 5.5: [cc lang=“bash” width=“100%” theme=“blackboard” nowrap=“0”] ~ # esxcli software profile update -d https://hostupdate.vmware.com/software/VUM/PRODUCTION/main/vmw-depot-index.xml -p ESXi-5.
