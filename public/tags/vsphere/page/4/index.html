One of the new features in vSphere 5.5 is the vSphere vFlash that enables you to use a SSD/Flash device as a read cache for your storage. Duncan Epping has a series of posts on vSphere Flash Cache that is well worth a read. vSphere vFlash caches your read IOs, but at the same time you can use it as a swap device if you run into memory contention issues.
